Fractional Trade Bot

Languges and Technologies used to create this project: C#, SQL, RESTful API's, XML, JSON.

The program uses 'Binance's' API marketplace to make informed decisions based on the price of the product, in this case, cryptocurrencies.

I created a system which reads in a product and it's current data asynchronously and based on its performance during a certain period of time, i.e. '1 minute', give it a rating from "1 -10" (1 being good, 10 being bad).

As the system is doing this, I also have another thread to manage 'buying'(in reality i'm placing a market order with some criteria, i.e. buy at this price, and sell once this crytpo reaches this price, the same goes for the other way around.) all the cryptocurrencies that my other system has deemed 'good' to 'satifactory'.

Once a buy order has been placed on Binance's marketplace, I record all successful buy orders into my database alongside information such as product ID, buy time and buy price, sell time and sell price, etc.

I also have a messaging system which sends me a message on things such as: Exceptions, Current Binance Wallet Balance, etc.

This project is still on going because of the size and security of the program.
